# 项目团队跳转功能实现说明

## 功能描述
在项目列表页面，点击"操作"列中的"团队"按钮，跳转到团队管理页面，并自动将该项目的负责团队ID填充到搜索栏中。

## 实现内容

### 1. 项目列表页面修改

**文件**: `ruoyi-ui/src/views/project/project/index.vue`

#### 修改 handleTeam 方法

```javascript
/** 团队按钮操作 */
handleTeam(row) {
  // 跳转到团队管理页面，并传递团队ID作为搜索条件
  if (row.teamId) {
    this.$router.push({
      path: '/project/team',
      query: {
        teamId: row.teamId
      }
    })
  } else {
    this.$modal.msgWarning("该项目未设置负责团队")
  }
}
```

**功能说明**：
- 检查项目是否有 teamId
- 如果有，跳转到团队管理页面并通过 query 参数传递 teamId
- 如果没有，提示用户该项目未设置负责团队

### 2. 团队管理页面修改

**文件**: `ruoyi-ui/src/views/project/team/index.vue`

#### 修改 created 钩子

```javascript
created() {
  // 从路由参数中获取 teamId
  if (this.$route.query.teamId) {
    this.queryParams.teamId = this.$route.query.teamId
  }
  this.getList()
}
```

**功能说明**：
- 在页面创建时检查路由参数中是否有 teamId
- 如果有，将其赋值给查询参数 queryParams.teamId
- 然后执行 getList() 查询，自动过滤出该团队的信息

## 使用流程

1. **在项目列表中**
   - 打开项目列表页面
   - 找到需要查看团队的项目
   - 点击"操作"列中的"团队"按钮

2. **自动跳转和过滤**
   - 系统自动跳转到团队管理页面
   - 搜索栏的"团队ID"字段自动填充为该项目负责的团队ID
   - 列表自动显示该团队的详细信息

3. **手动清除过滤**
   - 如需查看所有团队，点击"重置"按钮清除搜索条件

## 技术要点

### 1. Vue Router 参数传递
使用 `query` 参数传递数据：
```javascript
this.$router.push({
  path: '/project/team',
  query: {
    teamId: row.teamId
  }
})
```

### 2. 路由参数接收
在目标页面的 `created` 钩子中接收：
```javascript
if (this.$route.query.teamId) {
  this.queryParams.teamId = this.$route.query.teamId
}
```

### 3. 用户体验优化
- 如果项目没有设置团队，给出友好提示
- 自动填充搜索条件，减少用户操作
- 保留重置功能，方便用户查看所有团队

## 注意事项

1. **确保路由配置正确**
   - 团队管理页面的路由路径必须是 `/project/team`
   - 检查 `src/router/index.js` 中的路由配置

2. **teamId 数据类型**
   - 从路由参数获取的 teamId 是字符串类型
   - 如果需要数字类型，可以进行转换：`parseInt(this.$route.query.teamId)`

3. **权限控制**
   - 确保用户有访问团队管理页面的权限
   - 确保用户有查看团队列表的权限

## 扩展建议

如果未来需要传递更多信息，可以扩展 query 参数：

```javascript
this.$router.push({
  path: '/project/team',
  query: {
    teamId: row.teamId,
    teamName: row.teamName,  // 团队名称
    projectId: row.projectId  // 项目ID
  }
})
```

然后在团队管理页面接收并使用这些参数。
