# 项目负责团队功能实现说明

## 功能概述
为项目列表的新增和修改功能添加"负责团队"字段，通过关联 project_member 表显示团队名称。

## 实现内容

### 1. 数据库层面

#### 1.1 现有字段
- **project 表**：已有 `team_id` 字段，用于存储负责团队的ID
- **project_member 表**：需要添加 `team_id` 和 `team_name` 字段

#### 1.2 数据库迁移脚本
执行 `sql/add_team_fields.sql` 脚本，为 project_member 表添加以下字段：
```sql
ALTER TABLE project_member 
ADD COLUMN team_id BIGINT(20) DEFAULT NULL COMMENT '团队ID',
ADD COLUMN team_name VARCHAR(100) DEFAULT NULL COMMENT '团队名称';
```

### 2. 后端实现

#### 2.1 实体类修改
**文件**: `ruoyi-project/src/main/java/com/ruoyi/project/domain/Project.java`

添加了 `teamName` 字段及其 getter/setter 方法：
```java
/** 团队名称 */
@Excel(name = "团队名称")
private String teamName;

public void setTeamName(String teamName) {
    this.teamName = teamName;
}

public String getTeamName() {
    return teamName;
}
```

#### 2.2 Mapper XML 修改
**文件**: `ruoyi-project/src/main/resources/mapper/project/ProjectMapper.xml`

修改查询语句，通过 LEFT JOIN 关联 project_member 表获取团队名称：
```xml
<sql id="selectProjectVo">
    select p.project_id, p.project_name, ..., p.team_id, pm.team_name, ...
    from project p
    left join (select team_id, team_name, project_id from project_member group by team_id, project_id) pm 
    on p.team_id = pm.team_id and pm.project_id = p.project_id
</sql>
```

### 3. 前端实现

#### 3.1 项目列表页面
**文件**: `ruoyi-ui/src/views/project/project/index.vue`

##### 3.1.1 表格列添加
在项目列表表格中添加"负责团队"列：
```vue
<el-table-column label="负责团队" align="center" prop="teamName" :show-overflow-tooltip="true" />
```

##### 3.1.2 表单添加团队选择器
在新增/修改项目对话框中添加团队选择器：
```vue
<el-form-item label="负责团队" prop="teamId">
  <el-select v-model="form.teamId" placeholder="请选择负责团队" clearable style="width: 100%">
    <el-option
      v-for="team in teamOptions"
      :key="team.teamId"
      :label="team.teamName"
      :value="team.teamId"
    />
  </el-select>
</el-form-item>
```

##### 3.1.3 API 导入和数据加载
导入团队API并加载团队列表：
```javascript
import { listTeam } from "@/api/project/team"

// 在 created 钩子中加载团队列表
created() {
  this.getList()
  this.getTeamList()
},

// 获取团队列表方法
getTeamList() {
  listTeam({ pageNum: 1, pageSize: 1000 }).then(response => {
    this.teamOptions = response.rows || []
  })
}
```

##### 3.1.4 表单重置
在 reset 方法中添加 teamId 和 teamName 字段：
```javascript
reset() {
  this.form = {
    // ... 其他字段
    teamId: null,
    teamName: null,
    // ... 其他字段
  }
}
```

## 使用说明

### 1. 数据库初始化
执行 SQL 脚本：
```bash
mysql -u username -p database_name < sql/add_team_fields.sql
```

### 2. 重新编译后端
```bash
mvn clean package
```

### 3. 重启应用
重新启动后端服务使修改生效。

### 4. 前端使用
1. 打开项目列表页面
2. 点击"新增"或"修改"按钮
3. 在表单中选择"负责团队"
4. 保存后，项目列表中会显示团队名称

## 数据关系说明

- **project.team_id**: 项目的负责团队ID
- **project_member.team_id**: 团队成员所属的团队ID
- **project_member.team_name**: 团队名称

通过 project.team_id 关联到 project_member 表中相同 team_id 的记录，获取 team_name 进行显示。

## 注意事项

1. 确保在执行数据库迁移脚本前备份数据
2. team_id 和 team_name 字段允许为空，以兼容历史数据
3. 团队数据来源于 project_member 表，需要先有团队成员数据
4. 一个项目只能有一个负责团队
5. 团队选择器支持清空操作（clearable）
