📋 状态说明

📋 待办/未开始 (0) 任务已创建但尚未开始执行，可以开始执行或暂时搁置
⚙️ 进行中 (1) 任务正在执行中，可以完成、暂停或回退到待办状态
✅ 已完成 (2) 任务已经完成，如有需要可以重新打开进行修改
⏸️ 暂停 (3) 任务暂时停止执行，可以恢复执行或重新安排

💡 使用说明

符号说明：✓ 表示允许的流转   |   ✗ 表示不允许的流转

如何保存为图片：

1. 方法一（推荐）：右键点击上方的状态流转图 → 选择"复制图片" → 粘贴到 Word 或其他文档中

2. 方法二：使用截图工具（如微信截图 Alt+A 或 QQ 截图 Ctrl+Alt+A）截取图表区域

3. 方法三：点击下方的"打印"按钮 → 在打印对话框中选择"另存为 PDF" → 从 PDF 中提取图片

论文插入建议：建议将图片保存为 PNG 格式（透明背景），然后在论文中使用"插入图片"功能添加。
